# 快速编辑
## 准备
1. 下载编辑器 vscode, 
2. 安装vs插件 Markdown All in One
3. 安装vs插件 Paste Image
4. 重启编辑器
## 自动生成目录
按 *ctrl + shift + P* 打开命令面板， 输入框中输入 *ctc*, 执行创建目录命令；即可自动生成目录。
