## 类型
| 类型  |         描述         |
| :---: | :------------------: |
|   &   |        按位与        |
|  \|   |        按位或        |
|   ^   |       按位异或       |
|   ~   |        按位非        |
|  <<   |      带符号左移      |
|  >>   |      带符号右移      |
|  >>>  | 无符号右移(结果非负) |

## 总结
1. & 操作用来筛选1
2. | 操作用来筛选0
3. ^ 操作用来切换值，判定异同

## 应用
|      表达式      |              描述              |
| :--------------: | :----------------------------: |
|  a \| 0 或 ~~a   |            向下取整            |
|  a + 0.5 \|   0  |            四舍五入            |
|   a + 1 \|   0   |            向上取整            |
|      a << n      |      整数部分 x 2的n次幂       |
|      a >> n      | 整数部分 / 2的n次幂,并向下取整 |
|  ( a & 1 ) == 1  |           判断为奇数           |
| ( a & a-1 ) == 0 |       判定为2的整数次幂        |
|      a ^= 1      |          切换0，1的值          |
| a^=b; b^=a; a^=b |             交换值             |
|  ( a ^ b ) < 0   |            判定异号            |
