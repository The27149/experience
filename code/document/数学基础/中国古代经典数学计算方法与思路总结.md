# 中国古代经典数学计算方法与思路总结

核心思想：**化整为零，控制变量，分步求解，合并结果**

## 一、中国剩余定理（大衍求一术/韩信点兵）

### 适用题型

一个数除以不同互质数，余数各不相同，求满足条件的最小数

### 计算方法与步骤

1. 求两两互质的除数的**最小公倍数**

2. 构造专属数：分别求其他除数公倍数中，除以当前除数**余1**的数

3. 按题目余数，将专属数与对应余数相乘后求和

4. 用总和不断减最小公倍数，得到最小正整数解

### 经典实例

求：除以3余2、除以5余3、除以7余2的最小数

1. 3、5、7最小公倍数：105

2. 构造专属数：

    - 5和7的倍数÷3余1：70

    - 3和7的倍数÷5余1：21

    - 3和5的倍数÷7余1：15

3. 求和：70×2 + 21×3 + 15×2 = 233

4. 求最小解：233 - 105×2 = **23**

## 二、正负开方术（高次方程/开高次方）

### 适用题型

解任意高次方程、开任意次方（如开10次方、开立方）

### 计算方法与步骤

1. **估算首位**：确定方程根的整数部分

2. **拆分数值**：设根=已确定数值+微小增量

3. **二项式展开**：借助贾宪三角（杨辉三角）取展开式系数

4. **降次简化**：舍弃微小增量的高次项，转化为一元一次方程

5. **迭代精算**：求出增量后，将新结果作为基数，重复上述步骤逼近精确值

### 经典实例

开10次方求 $x^{10}=N$ 

1. 估算整数位a，设 $x=a+t$ （t为微小增量）

2. 借助贾宪三角第11行，展开 $(a+t)^{10}$ 

3. 保留前两项： $a^{10}+10a^9t≈N$ 

4. 求解 $t=(N-a^{10})/(10a^9)$ ，循环迭代得精确值

### 关键工具

贾宪三角（杨辉三角）：两侧数字为1，中间数=肩上两数之和，直接获取高次展开系数

## 三、三角形相关定理（代数+几何通用）

### （一）三斜求积术（秦九韶公式）

#### 适用题型

已知三角形三条边长，求三角形面积

#### 计算方法

设三角形三边为小斜 $a$ 、中斜 $b$ 、大斜 $c$ ，面积公式：

 $S=\sqrt{\frac{1}{4}\left[ a^2c^2-\left( \frac{a^2+c^2-b^2}{2} \right)^2 \right]}$ 

#### 经典实例

三角形三边3、4、5，代入公式计算得面积 $S=6$ 

### （二）海伦公式

#### 适用题型

已知三角形三边，纯代数计算面积（无需角度/高）

#### 计算方法与步骤

1. 求半周长： $p=\frac{a+b+c}{2}$ 

2. 面积公式： $S=\sqrt{p(p-a)(p-b)(p-c)}$ 

#### 经典实例

三边3、4、5， $p=6$ ， $S=\sqrt{6×3×2×1}=6$ ，与三斜求积术结果等价

### （三）正弦定理

#### 适用题型

已知三角形边角关系，求未知边/未知角

#### 定理内容

 $\frac{a}{\sin A}=\frac{b}{\sin B}=\frac{c}{\sin C}=2R$ （ $R$ 为三角形外接圆半径）

#### 解题思路

边角互化，利用对角与对边的正比例关系求解

#### 经典实例

△ABC中， $a=2$ ， $∠A=30°$ ， $∠B=60°$ ，由 $2R=\frac{2}{\sin30°}=4$ ，得 $b=4\sin60°=2\sqrt{3}$ 

### （四）余弦定理

#### 适用题型

已知两边夹角求第三边、已知三边求任意内角

#### 定理内容

 $a^2=b^2+c^2-2bc\cos A$ 

 $b^2=a^2+c^2-2ac\cos B$ 

 $c^2=a^2+b^2-2ab\cos C$ 

#### 解题思路

将角度关系转化为边长代数关系，实现边角互推

#### 经典实例

三边3、4、5，求最长边对角： $\cos A=\frac{3^2+4^2-5^2}{2×3×4}=0$ ，得 $∠A=90°$ 

### （五）射影定理（三角形边角投影关系）

#### 适用题型

边角相互推导，可推导正余弦定理

#### 定理内容

 $a = b\cos C + c\cos B$ 

 $b = a\cos C + c\cos A$ 

 $c = a\cos B + b\cos A$ 

### （六）三角形面积通用公式

#### 适用题型

已知两边及夹角，求三角形面积

#### 公式内容

 $S=\frac{1}{2}ab\sin C=\frac{1}{2}bc\sin A=\frac{1}{2}ac\sin B$ 

### （七）三角形常用边角恒等式

1. 内角和： $A+B+C=\pi$ 

2. 诱导公式： $\sin A=\sin(B+C)$ ， $\cos A=-\cos(B+C)$ 

3. 正切恒等式（非直角三角形）： $\tan A+\tan B+\tan C=\tan A·\tan B·\tan C$
